Context: the purpose of this project is to pick a website of our choosing that we would like to improve. I chose a website for a township in PA. As you will see below, the website is quite outdated looking and is not responsive to different screen sizes. I believe I can revamp their site to make it more responsive and modernized.Picking a webpage: I chose to redesign Bristol Township’s webpage. As you can see above, the page is quite outdated. http://www.bristoltownship.org/index.cfm

 

Finding Problems:

  • Usability (like efficiency)
  • This website lacks efficiency because it doesn’t have a clear hierarchy of information, nor does it have a search bar at the top of the page. The webpage does not direct the user’s eyes to key information. Instead, users are left with figuring out key information for themselves and using a significant amount of cognitive effort sorting through the large amount of text.
  • Learnability
  • At first glance it is not clear how to navigate certain blocks of text. For example, there is no clear indication that certain lines of text are actual links that take you to another page. If you look on the left hand side in the blue column, you’ll see the Bristol Township logo with the different TV channels below and then a LIVE STREAM. However, it is unclear that the LIVE STREAM text is actually a link because the text is the same color as the rest of the text around it.
  • I think the learnability is generally quite low on this site due to the lack of organization and style choices.
  • Memorability
  • The memorability for this webpage is lacking. It doesn’t use any visual cues/icons to indicate more information. Visual cues are essential in developing a mental model of something. Overall, the website is not very impressionable. Thus, someone returning to the website after not using it for awhile may struggle to navigate the site and find key information.
  • Usability issue and mental model issue:Can't tell that these dates are actually links that link to a pdf. Also, when you click on it, it opens the pdf on the same window instead of a new window which is an issue for some.

Accessibility:

  • The WAVE assessment pointed out many issues with the site. To begin, there isn’t any alt text for the images. WAVE also pointed out that it features redundant links. For example, each council person has a hyperlink on their name, yet each of those links goes to the same page that displays all the council people in further detail. Furthermore, the site lacks any ARIA labels, but this is likely due to the fact that the website doesn’t use many visual elements (like hamburger menus). The final main issue that WAVE pointed out was that the NEWS column on the right uses justified text. This is a problem for readability due to the varying word/letter spacing.
